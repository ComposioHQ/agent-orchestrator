# Design Mode Slot

This slot holds configurable design strategies for the `design-studio` agent.

Each config selects:

- a prompt source
- the design artifact mode
- brand and copy focus areas
- deliverables for the first design pass
- principles and anti-patterns that keep the work sharp

Current configs:

- `codex.json`: brand-system, UI, and landing-page direction tuned for ao/codex-style startup launches
